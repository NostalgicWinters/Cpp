# C++ Notes & DSA

A personal repository containing my C++ notes, implementations, and Data Structures & Algorithms practice.

This is my go-to reference whenever I want to revise C++, understand a concept again, or review common DSA implementations. Instead of searching through multiple websites or videos, I keep everything I learn in one place.

## Contents

* C++ fundamentals
* Object-Oriented Programming
* STL (Standard Template Library)
* Algorithms
* Data Structures
* Problem-solving patterns
* Practice implementations
* Miscellaneous notes and experiments

## Purpose

This repository is not meant to be a complete course or a polished library. It serves as my personal knowledge base for:

* Revising C++ concepts
* Preparing for coding interviews
* Competitive programming
* Academic coursework
* Quickly looking up syntax or implementations
